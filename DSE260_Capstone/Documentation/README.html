Overview

This is the README file for the UCSD DSE capstone project for Team 3 “Drive Classy, San Diego”.

To see the API documentation and the procedures to follow to load the data and run the code, see Documentation/docs/index.html.

The programs are written in Python 2.7, and PostgreSQL,was used as the database along with the PostGIS spatial extensions. Documentation was created using Sphinx. Version control was stored in Git repository on github.
